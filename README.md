## Run Locally

Clone the project

```bash
  git clone https://github.com/bodln/Pytnik-Backend
```

Go to the project directory

```bash
  cd Pytnik-Backend
```

Install dependencies

```bash
  pip install
```

Go to the folder that contains manage.py

```bash
  cd pytnik-backend
```

Start the server

```bash
  py .\manage.py runserver
```
